## Запуск dev версии
```
git clone https://github.com/streljk93/hhru.git
cd hhru
npm install
npm run-script dev
```

## Запуск production
```
git clone https://github.com/streljk93/hhru.git
cd hhru
npm install
npm run-script build
npm start
```
> Далее заходим на localhost:8080
